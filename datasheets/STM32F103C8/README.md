# STM32F103C8 Documentation Index

This directory groups the key documentation for the `STM32F103C8` target.

Target identified on the bench:

- family: `STM32F1`
- device class: `STM32F10x medium-density`
- exact part confirmed by user: `STM32F103C8`

## Notes

- These are the relevant documents for bring-up, flash backup/restore, UART use, and ROM bootloader use on `STM32F103C8`.
- Direct PDF downloads from `st.com` were not reachable from this Raspberry Pi at the time this index was created, so this directory currently stores a curated local index of the official documents rather than local PDF copies.
