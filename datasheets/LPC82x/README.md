# LPC82x Documentation Index

This directory holds the local reference set for the NXP `LPC82x` family.

For current bench work, the most relevant device is:

- `LPC824`

This local set is family-level, so it covers `LPC824` through the shared
`LPC82x` datasheet, user manual, and errata.

## Core Documents

- `LPC82X.pdf`
  - NXP `LPC82x` family datasheet
  - primary device datasheet for `LPC824`
- `UM10800.pdf`
  - NXP `LPC82x` user manual
  - primary peripheral/register reference for `LPC824`
- `ES_LPC82X.pdf`
  - NXP `LPC82x` errata

## Application Notes / Software Packages

- `AN13815.pdf`
  - USART ISP programming note for `LPC800` family parts
- `AN12726.pdf`
  - SCTimer/PWM application note
- `AN11782.zip`
  - NXP software package related to `LPC82x` family use
- `AN12726SW.zip`
  - NXP software package accompanying `AN12726`
- `AN13095SW.zip`
  - NXP software package for `LPC82x` family examples/support
